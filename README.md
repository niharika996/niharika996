## 📌 Featured Projects

### 🧠 Corporate Brain – Enterprise RAG System *(Ongoing)*
> Secure, role‑based RAG pipeline for private document Q&A with hybrid search, agentic reasoning, and RBAC.
- **Stack:** Python, FastAPI, Qdrant, Elasticsearch, PostgreSQL, Redis, Groq LLM, Docker  
- Hybrid search (RRF) + cross‑encoder reranking  
- Self‑reflection loop (Plan → Act → Verify) – confidence threshold 0.55  
- RBAC enforcement at search level (admin/hr/finance/general)  
- Multi‑format ingestion (PDF, DOCX, XLSX, PPTX) + OCR for scanned PDFs  

---

### 🌆 City-Shield – Urban Risk Intelligence System
> AI‑driven early warning system for urban flooding, heatwaves, and air pollution.  
**My role:** Heatwave prediction module using Landsat 8 thermal data + OpenWeather API.  
- Physics‑Aware Digital Twin (30m resolution)  
- Ward‑level risk classification (Low → Critical)  
- Team: flood mapping (SAR + weak supervision), air pollution (LSTM + quantile regression), scenario simulation dashboard  
- **Stack:** Python, TensorFlow, Landsat/Sentinel, React, MapLibre GL, SHAP  

---

### 📱 AdaptLearn AI
> Offline‑first Android app that helps children with dyslexia read using real‑time Edge AI.  
- **Three‑packet design:** CameraX + ML Kit OCR → Llama.cpp (on‑device) → Jetpack Compose UI  
- Struggle detection (face tracking + audio pauses >2s)  
- 100% on‑device – no internet required, zero data leakage  
- **Stack:** Kotlin, Jetpack Compose, llama.cpp, ML Kit, CameraX, Room DB, Coroutines
